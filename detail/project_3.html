Full DevOps Automation Project

Project Informatie
  • Technologie: Git, Docker, Kubernetes, Jenkins
  • Duur: 8 weken
  • Status: Afgerond

Project Details

Voor het olod DevOps werd een doorlopend project uitgevoerd waarbij alle geziene leerstof werd geïntegreerd in één complete workflow. De opdracht vertrok vanuit een webapplicatie bestaande uit minstens drie componenten: een frontend, een backend en een database.

Alle onderdelen van de applicatie werden verpakt in containers met behulp van Docker. De deployment, containerlinks en persistente opslag werden gedefinieerd met een orchestration tool, zoals Docker Compose of Kubernetes.

De volledige applicatie en alle DevOps-code werden beheerd in een online Git-repository. Vanuit deze repository werd een geautomatiseerde CI/CD-pipeline opgezet met Jenkins, waardoor de volledige applicatie met één druk op de knop gedeployed kan worden.

Het project omvatte daarnaast duidelijke documentatie in een README.md-bestand en correct versiebeheer via regelmatige commits en pushes.
